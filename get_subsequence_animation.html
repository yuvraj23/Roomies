Problem Setup

Code
public static ArrayList<String> getSS(String que) { if(que.length() == 0) { ArrayList<String> ans = new ArrayList<>(); ans.add(""); return ans; } char firstChar = que.charAt(que.length()-1); que = que.substring(0, que.length()-1); ArrayList<String> list = getSS(que); ArrayList<String> wholeAns = new ArrayList<>(); for(String s : list) { wholeAns.add(s + firstChar); } for(String s : list) { wholeAns.add(s + "."); } return wholeAns; }

Function Explanation

Code
public static ArrayList<String> getSS(String que) { if(que.length() == 0) { ArrayList<String> ans = new ArrayList<>(); ans.add(""); return ans; } char firstChar = que.charAt(que.length()-1); que = que.substring(0, que.length()-1); ArrayList<String> list = getSS(que); ArrayList<String> wholeAns = new ArrayList<>(); for(String s : list) { wholeAns.add(s + firstChar); } for(String s : list) { wholeAns.add(s + "."); } return wholeAns; }
Key Steps
  1. Base case: Empty string returns [""]
  2. Extract the last character from the string
  3. Make recursive call with the shorter string
  4. For each subsequence from the recursive call:
    • Create a version that includes the character
    • Create a version that excludes the character

The function works by including or excluding each character

Step 11/12

Key Insights

Code Patterns
// Base case if(que.length() == 0) { ArrayList<String> ans = new ArrayList<>(); ans.add(""); return ans; } // Recursive breakdown char firstChar = que.charAt(que.length()-1); que = que.substring(0, que.length()-1); ArrayList<String> list = getSS(que); // Include/exclude pattern for(String s : list) { wholeAns.add(s + firstChar); // Include } for(String s : list) { wholeAns.add(s + "."); // Exclude }
Key Insights
  • Recursion breaks down the problem by processing one character at a time
  • The include/exclude pattern creates all possible combinations
  • The number of subsequences is 2^n where n is the string length
  • The call stack tracks the state at each level of recursion
  • Base case prevents infinite recursion and provides the starting point
